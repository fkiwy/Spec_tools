System requirements: Any FITS capable reader.

Additional comments: Each NIRES spectrum has been shifted to the rest frame 
 wavelength, and NIRES order has been scaled to the low-resolution spectrum 
 and offset by a constant value (see FITS headers for details). Each 
 low-resolution spectrum has been normalized to its median flux 
 between 1.2-1.25 um. Spectra have also been shifted to the rest frame 
 (see text). Copies for each source in .csv format are also included.
